# DS Summer 2026

This is my learning repository for May–August 2026.

I'm a 5th-semester Data Science student at the University of Toledo. Until 
now, most of my "projects" were built with heavy AI assistance — I'd ask 
ChatGPT or Claude to architect things and I'd follow along without really 
understanding the code. I'm done with that.

This summer I'm restarting from the foundations and learning data science 
properly. The goal is to actually understand what I'm doing — to be able 
to defend every decision in any technical interview.

## What's in here

- `daily-logs/` — short daily entries: what I learned, what confused me, 
   what I couldn't explain back to myself
- `notes/` — section-by-section notes from courses I'm taking
- `code/` — all the code I write while learning (notebooks, scripts, 
   exercises)
- `tests/` — weekly knowledge tests and my answers
